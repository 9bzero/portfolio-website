# Portfolio Website

Personal developer portfolio — dark theme, smooth animations, and a curated project showcase.

## Features

- Dark theme with electric accent colors
- Smooth entrance animations (Framer Motion)
- Live GitHub project feed via GitHub API
- Category filter: Dev Tools, Games, Visualizations, Design Tools, Calculators, Productivity
- Project cards with description, topics, and GitHub links
- Tech stack showcase
- Responsive layout
